## 🔧 Adding More Templates

### Template Structure:

```typescript
// src/services/templates/category/name.ts

export const html = `...`;
export const css = `...`;
export const javascript = `...`;

export default { html, css, javascript };
```

### Registration:

```typescript
// src/services/enhancedTemplateService.ts

// 1. Import template
import templateName from './templates/category/name';

// 2. Add to registry
const templateRegistry = {
  'category-name': templateName,
};

// 3. Add metadata
const templateMetadata = {
  'category-name': {
    id: 'category-name',
    name: 'Template Name',
    category: 'category',
```
